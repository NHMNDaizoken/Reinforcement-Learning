# BTCK — Smart Traffic Signal Digital Twin

## What This Is

Multi-Agent RL system for traffic signal control. 9 intersections in a 3×3 CityFlow grid, each is an independent DQN agent. All agents share one neural network (Parameter Sharing). Trained with MaxPressure reward. Compared against a rule-based MaxPressure baseline.

Algorithm: **PressLight** (Independent DQN + Parameter Sharing + MaxPressure reward).

## Key Design Decisions

- **CityFlow** over SUMO: 10–50× faster, native Python API, used in PressLight paper.
- **Parameter Sharing**: all 9 agents share `θ_shared`. Sample efficiency ×9.
- **MaxPressure reward** `r_i = -(Σq_in - Σq_out)`: theoretically grounded, O(1) compute, no heuristic weight tuning.
- **Huber loss** instead of MSE: prevents gradient explosion during heavy congestion.
- **buffer_size=5000, batch=64**: sweet spot for a 14-day sprint. Larger values need more training time.
- **Offline dataset** saved to SQLite with schema `(episode, step, agent_id, state_vec, action, reward, next_state_vec, done, atl, throughput)` — precomputed `atl` and `throughput` per row avoids re-running the simulator for the dashboard.

## Core Formulas

**State:** `s_i = [q_in_1..q_in_k, q_out_1..q_out_k, one_hot(phase)]`

**Reward:** `r_i = -(Σq_in_l - Σq_out_m)` (negative MaxPressure)

**Target:** `y_j = r_j + γ · max_a' Q(s'_j, a'; θ⁻)`

**Loss:** Huber(`y_j - Q(s_j, a_j; θ)`)

## Running the Pipeline

```bash
# 1. Generate configs
python configs/generate_configs.py

# 2. Train DQN (500 episodes, medium flow)
python src/training/train_dqn.py --roadnet configs/roadnet.json --flow configs/flow_medium.json

# 3. Run baseline + merge offline dataset
python src/evaluate.py --episodes 50

# 4. Generate analysis plots
python analysis/plot.py

# 5. Start API
uvicorn api.main:app --reload --port 8000

# 6. Start frontend
cd web && npm run web
```

## Success Criteria

DQN after training must beat the MaxPressure Baseline on:
- **Average Travel Time (ATL)** — lower is better
- **Throughput** — higher is better

On the High (900 veh/h) scenario, the gap should be clearly visible.
